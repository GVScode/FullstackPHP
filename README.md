
# 🚀 Curso Fullstack PHP & Laravel

Bem-vindo ao repositório oficial do **celke Fullstack PHP**! Este repositório contém todo o material prático, códigos das aulas, exercícios e o projeto final desenvolvido ao longo da formação. O objetivo é levar o aluno do zero ao deploy de uma aplicação web completa e profissional.

---

## 📚 O que você vai aprender (Módulos)

1. **Fundamentos Modernos do PHP:** Sintaxe, Orientação a Objetos (OOP) e Composer.
2. **Introdução ao Laravel:** Arquitetura MVC, Rotas, Controllers, Requests e Views (Blade).
3. **Banco de Dados e Eloquent ORM:** Migrations, Seeders, Factories, Relacionamentos e Query Builder.
4. **Frontend Integrado:** Construção de interfaces responsivas com Tailwind CSS, Vite e Alpine.js (ou Vue.js/React, conforme o curso).
5. **Autenticação e Segurança:** Laravel Breeze/Jetstream, Middleware, Gates e Policies.
6. **APIs RESTful:** Criação de endpoints, consumo via JavaScript e Laravel Sanctum.
7. **Qualidade de Código:** Testes automatizados com Pest ou PHPUnit.
8. **Deploy e DevOps:** Preparação para produção, Docker (opcional) e boas práticas de CI/CD.

---

## 🛠️ Tecnologias Utilizadas

- **Backend:** PHP 8.2+, Laravel 11.x
- **Frontend:** Blade, Tailwind CSS, Vite, Alpine.js
- **Banco de Dados:** MySQL / PostgreSQL
- **Ferramentas:** Git, Composer, Node.js, Postman/Insomnia

---

## ⚙️ Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina:
- [PHP](https://www.php.net/) (versão 8.2 ou superior)
- [Composer](https://getcomposer.org/)
- [Node.js](https://nodejs.org/) (versão 18 ou superior) e NPM
- Um servidor de banco de dados (MySQL, MariaDB ou PostgreSQL)
- [Git](https://git-scm.com/)
